# Частотный взгляд на нейросети: F-принцип в практике

Практическая часть курсовой работы, посвящённая исследованию **F-принципа** (Feature-following principle) в обучении нейронных сетей через анализ частотной динамики.

## Описание проекта

Этот репозиторий содержит эксперименты, которые демонстрируют:

-  **Проверку F-принципа** на синтетических примерах
-  **Влияние признаков Фурье** на ускорение обучения высоких частот
-  **Спектральный анализ обучения** нейронных сетей
-  **Влияние архитектуры сети** (ширина, глубина) на динамику спектра
-  **Влияние гиперпараметров** (скорость обучения) на обучение

## Цели работы

1. На практике проверить наличие F-принципа на синтетических примерах
2. Установить, насколько сильно признаки Фурье ускоряют обучение высоких частот
3. Исследовать влияние ширины сети и шага обучения на спектральные кривые
